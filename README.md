## 📱 Project Overview

**Pattern & Sign Language (باترون وإشارة)** is an innovative, native educational application built with Flutter. It aims to bridge fashion design with sign language, empowering the deaf and hard-of-hearing community to learn garment construction and pattern-making seamlessly.

Unlike standard tutorial platforms, it features an accessible, user-centric environment with **translated sign-language video guides**, **interactive pre/post tests**, and an **interactive feedback community**.

---

## ✨ Key Features

### 🤟 Accessible E-Learning
* **Sign Language Integration:** Practical video lessons embedded directly with translated sign language.
* **Dual Evaluation Engine:** Interactive pre-test (اختبار قبلي) and post-test (اختبار بعدي) to assess skill progress.
* **Visual Measurement Guides:** Step-by-step children's garment drafting and body measurement charts.

### 💬 Interactive Community & Progress
* **Feedback Hub:** Discussion board where users can comment, post work samples, and ask questions.
* **Session Tracking:** Progress tracking with completion percentages and lesson status toggles.
