# 🪡 Pattern & Sign Language App | باترون وإشارة 🤟

An innovative, educational mobile application designed to bridge fashion design with sign language, empowering the deaf and hard-of-hearing community to learn garment construction and pattern making.

---

## 📱 Project Overview | نبذة عن المشروع

**Pattern & Sign Language** is an inclusive educational platform that breaks accessibility barriers in vocational training. By integrating specialized fashion design content with translated sign language videos, the application enables self-paced learning for deaf individuals without needing an in-person instructor.

---

## ✨ Key Features | المميزات الرئيسية

* 🤟 **Sign Language Integration:** Instructional videos accompanied by clear, translated sign language for deaf learners.
* ✂️ **Pattern Making & Sewing Skills:** Step-by-step simplified lessons on garment drafting and tailoring for children's wear.
* 📐 **Interactive Measurement Guides:** Visual and structured tutorials explaining precise body measurement techniques.
* 📚 **Theoretical & Practical Content:** A balanced curriculum covering foundational knowledge alongside practical application.
* 🎨 **User-Centric UI/UX:** Clean, intuitive interface tailored for an accessible and smooth learning experience.

---

## 🛠️ Tech Stack | التقنيات المستخدمة

* **Framework:** [Flutter](https://flutter.dev/) (Cross-platform)
* **Language:** [Dart](https://dart.dev/)
* **State Management:** Provider / Bloc *(اختر المستعمل في مشروعك)*
* **Backend / Services:** Firebase *(لو مستخدمه)*
